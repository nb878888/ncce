# QQ 农场更新日志

本文档用于记录 `cwser/qq-farm-bot-private` 私有维护版的功能更新、修复和维护记录。

## 2026-07-06

- 修复对好友农场的操作。
- 新增“优先种植 2×2 作物”策略并默认开启：检测到背包中有四格种子时，自动预留完整的 `2×2` 土地区域，等待四块土地全部空闲后优先种植。
- 四格作物仅使用背包库存，不会尝试从商城购买；预留区域之外的空地继续执行原有普通作物种植策略。
- 按实际抓包协议发送 `[左下、右下、左上、右上]` 四块土地，并校验服务器返回的主地块与从属地块关系。
- 已完整空闲的 `2×2` 土地可连续种植多颗四格种子；尚未清空的土地同一时间最多预留一组，避免长期占用大量普通种植区域。
- 登录首次巡田时即使没有空地，也会提前为四格种子规划一组土地；候选区域按预计整体清空时间选择，优先等待时间更短的组合。
- 多块土地同时空闲时使用最大不重叠组合选择，尽量种下更多四格作物；土地推送或策略刷新后会重新评估预留区域。
- 四格作物种植失败后等待 30 秒再重试，避免巡田间隔内重复请求和刷屏。
- 四格作物优先种植遇到当前种子锁定、未解锁或等级不足时，会自动切换其他可用 `2×2` 背包种子；若没有可切换种子，会释放本轮预留土地，避免普通种植被空占。
- 自动捣蛋连续失败 3 次后不再关闭前端开关，改为暂停到次日并由每日任务自动重新尝试；暂停日期会持久化，重启项目不会提前执行。
- 新账号的默认种植策略调整为“背包种子优先”，背包种子耗尽后继续使用配置的第二优先策略。
- 策略设置的“背包种子优先顺序”支持将单个种子移出优先列表，并可通过“重置顺序”恢复全部背包种子。
- 修复商城页移动端 tab 被排序、刷新按钮挤压的问题；保留活动页和商城页的横向 tab 适配，其余页面 tab 恢复原样。
- 前端运行日志和账号日志隐藏 ACE/TSDK 诊断信息，避免安全运行时初始化、上报、停止等内部日志干扰系统日志展示。
- 修复部分账号切换后 Dashboard 仍订阅旧账号实时日志，导致前端看不到当前账号日志的问题。
- 优化前端网络错误提示和后端 HTTP keep-alive 设置，减少切换页面时因瞬时连接复用或超时造成的误报。
- 修复后端请求超时保护与慢接口响应重复发送导致 `ERR_HTTP_HEADERS_SENT` 并退出的问题。
- 优化果实出售日志：批量出售失败后逐个重试时，不可售物品会单独记录跳过，最终出售成功日志只汇总真正卖出的物品和数量。

## 2026-07-04

- 新增“最终阶段普通肥”和“最终阶段有机肥”施肥策略；选择最终阶段普通肥时，多季补肥开关即使开启也不会触发补肥。
- 新增种子、化肥购买数量选择，支持手动输入数量和按余额计算最大可购买数量。
- 新增放虫放草连续失败三次后自动停止捣乱，并记录失败计数与停止日志。
- 优化金币、点券、金豆豆显示单位：金币超过 1 亿显示为“亿”，点券和金豆豆超过 1 万显示为“万”。

## 2026-07-03

- 优化移动端主布局滚动体验，修复顶部栏在部分页面遮挡内容的问题。
- 修复管理员卡密创建弹窗在手机端内容超出屏幕后无法上下滑动的问题。
- 同步优化管理员用户续费、编辑弹窗的小屏滚动表现，避免弹窗高度超过屏幕时卡住。

## 2026-07-02

- 修改 UI。
- 修复头像显示。
- 修复经验满后只帮助护主犬的问题。
- 新增“青梅酿万金”活动入口。
- 支持领取青梅种子，并根据抓包补齐领取参数。
- 新增青梅酿酒售卖流程，支持精酿、分享翻倍与售卖。
- 修复青梅酿酒缺少酿造记录导致的 `1034027` 错误。
- 优化活动页移动端 tab 栏，避免“青梅酿万金”在手机上超出屏幕。
- 优化右上角账号切换显示，展示当前账号名称并支持头像字段。
- 去除活动 tab 上不需要的数字徽标。

## 维护说明

- 运行时账号数据、日志、缓存和 `.env` 文件不进入仓库。
- 更新功能时优先参考实际抓包与当前代码结构，避免硬编码账号私有数据。
